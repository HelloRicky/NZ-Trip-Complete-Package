## 省钱技巧 💡

### 出发前
- [ ] 提早预订住宿（更好的价格）
- [ ] 寻找组合优惠（缆车 + 滑板车套餐）
- [ ] 提前在线预订活动（有时更便宜）
- [ ] 考虑旅行保险自付额以降低保费
- [ ] 下载离线地图（节省数据费用）

### 旅行期间
- [ ] 在超市购物（Pak'nSave最便宜）
- [ ] 自己做早餐（每人每天节省$15-20）
- [ ] 为一日游准备午餐
- [ ] 在小镇加油（通常更便宜）
- [ ] 免费活动：海滩、步道、游乐场、博物馆
- [ ] 自来水可以安全饮用（带可重复使用的水瓶）
- [ ] 寻找儿童免费用餐优惠
- [ ] 在餐厅分享家庭餐

### 值得花钱 vs 省钱
**值得花钱：**
- 观鲸（一生一次！）
- 天际缆车 + 滑板车（孩子们永远记住）
- 在皇后镇吃一顿特别的晚餐

**容易省钱：**
- 跳过付费热水池（自然风景免费！）
- 自己准备零食而不是在景点购买
- 驾驶而不是飞行皇后镇-基督城（节省约$700）
