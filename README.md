# 🇳🇿 NZ South Island Trip 2026 | 新西兰南岛之旅 2026

**April 11–20, 2026 · 7 travelers · 3 accommodation bases · JUCY Big Kahuna campervan**

This is a Next.js web application for planning our New Zealand South Island family trip.

---

## Getting Started | 开始使用

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

The app supports **English and Chinese** — toggle with the language button in the navbar.

---

## App Pages | 页面

| Page | EN | ZH |
|------|-----|-----|
| `/` | Trip overview, quick stats | 行程概览 |
| `/itinerary` | Day-by-day schedule | 每日行程 |
| `/map` | Interactive Leaflet map | 互动地图 |
| `/accommodation` | 3-base strategy | 住宿计划 |
| `/activities` | Bookings tracker | 活动与预订 |
| `/budget` | Cost breakdown | 预算追踪 |
| `/packing` | Checklist (saves to browser) | 打包清单 |
