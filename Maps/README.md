2. OR double-click `trip-map-offline.html` (for English)
3. Opens in your browser
4. No internet needed!

### To Share with Chris's Family | 与Chris一家分享:
1. Read `SHARING-PACKAGE.md` for instructions

## 🎯 Map Features | 地图功能

### What You Can See | 可以看到什么:

**Day Numbers | 天数标记:**
- Each location shows which day (1-10)
- Red circles = Accommodation bases
- Orange circles = Day trip destinations
- Blue circles = Special locations (Day 10 Akaroa)

**Daily Itinerary Sidebar | 每日行程侧边栏:**
- Complete 10-day schedule
- Activities for each day
- Flight times
- Accommodation info

**Interactive | 互动功能:**
- Click on day numbers to see details
- Click on day in sidebar to focus map
- Hover over locations for quick info

## ❓ Questions? | 有问题？

**Can't open the map?**
- Make sure you're using a web browser
- Try right-click → Open with → Chrome/Safari/Edge
- 确保使用网页浏览器
- 尝试右键 → 打开方式 → Chrome/Safari/Edge

**Need help sharing?**
- Check `SHARING-PACKAGE.md` for detailed instructions
- Email templates included
- 查看`SHARING-PACKAGE.md`获取详细说明
- 包含邮件模板

**Map not working on phone?**
- Download the file completely first
- Try opening in different browser (Chrome, Safari)
- 先完全下载文件
- 尝试在不同浏览器中打开（Chrome、Safari）
