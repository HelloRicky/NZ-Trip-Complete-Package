## 🔧 Troubleshooting | 故障排除

**If the map doesn't open: | 如果地图无法打开：**
1. Make sure you're opening it with a browser (Chrome, Safari, Edge, Firefox)
2. Try right-click → "Open with" → Choose browser
3. 确保用浏览器打开（Chrome、Safari、Edge、Firefox）
4. 尝试右键 →"打开方式"→ 选择浏览器

**If it looks blank: | 如果看起来是空白的：**
1. Wait a few seconds for it to load
2. Refresh the page (F5 or Cmd+R)
3. 等待几秒钟加载
4. 刷新页面（F5或Cmd+R）

**On mobile: | 在移动设备上：**
1. Make sure the file is fully downloaded
2. Try opening in different browsers
3. Some phones may need "Allow file access" permission
4. 确保文件已完全下载
5. 尝试在不同浏览器中打开
6. 某些手机可能需要"允许文件访问"权限
